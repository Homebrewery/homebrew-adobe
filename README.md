_[Homebrew]_ Adobe
==================
Homebrew [_Cask_]s and [formula]e for [Adobe] apps, and even [plugins]!

Install
-------
~~~ sh
brew tap danielbayley/adobe
brew cask install #cask
brew install #formula
~~~
